## [1.0.0] - 2025-01-17

### Added
- Initial release of Parchment-Assist Chrome extension
- AI-powered command suggestions for Z-machine interactive fiction games
- Dual AI backend support:
  - Local inference with Ollama
  - Cloud inference with Google Gemini
- Real-time game state tracking
  - Turn counter
  - Location mapping
  - NPC profiling
  - Inventory management
- Smart command suggestions based on game context
- Privacy-focused design with local-first option
- Comprehensive test suite:
  - Unit tests for core components
  - Integration tests for AI backends
  - Feature tests for game state management
- Developer tooling:
  - ESLint for code linting
  - Prettier for code formatting
  - Jest for testing with coverage reporting
- Documentation:
  - Architecture guide
  - Installation guide
  - Contributing guidelines
